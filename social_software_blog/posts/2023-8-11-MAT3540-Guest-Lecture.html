UXN Analysis

UXN is basically a operating system emulator inside a small virtual machine. Its purpose is to emulate the operating systems' commands and widgets that usually require some level of internet access in an offline space. The way the UXN is designed allows people to pull it up on a variety of devices not just a desktop and it works like an old game emulator that loads game roms, just instead of just games it can load things like digital clocks that have been converted to rom format.

The concept of the project prefectly describes what it felt like when I was messing around with the different available roms. It felt like I opened an old window from an old computer to load a program. I couldn't try it offline so I don't know how that aspect of the project feels but from my experience so far I think the emulation aspect works great.

What about the concept do you find compelling?

The concept in general is very compelling because its the offline internet that my chronically online self has always wanted when I was younger and the internet would just shut off. I like the idea that I could load the copies of web pages or things that I like if I can figure out how to rom it and run it through this system.

What are your critiques of the platform/project?

I feel like I haven't used it enough to make valid critiques yet. For what I have used and seen the platform works as intended and can be malleable enough for people to adapt what they need to adapt to the platform to run it. Honestly I'm not sure of the limits of the program but the only limit I've seen so far is the graphics and even then I like classic computer icons and graphics so its not a real detriment to the project. Overall review is 10/10 from me.

What alterations would you make to this software?

If I had the paitience the only thing I would change was how easy it is to load the roms and use them in a browser. I've only used it in a browser at this point and the only thing I didn't enjoy was constantly opening my files to pull up the rom to drag it into the browser. I would try to make like an in browser library for the roms or something. Like a extension that automatically pulls the rom folder so I just have to choose from there from inside the browser instead of using the console line or opening the files. But otherwise I had a great time using the roms that worked as intended. I hope to be able to load this on my DS or some other console at some point.
